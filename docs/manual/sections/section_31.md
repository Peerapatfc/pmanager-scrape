# 31. PMDD
PMDD, abbreviation of Planetarium Manager Developer Depot, consists of programs created to assist and aggregate all the developers who successfully create external tools to support Planetarium Manager users managing their teams.

XML feeds are used to access PM data.

If you are interested in developing such a tool, please create a PMDD ticket by going to Help >> Contact Us then submit a ticket describing your interest.
We will give you access to a private forum where you can discuss with other PM developers about your tool, and also obtain support on how to use the XML feeds.
