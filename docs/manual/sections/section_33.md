# 33. PManager Week
PManager is played in real-time, so each day of the week we'll have several different things happening. (All times are server-time)

Monday
- Stats database updates at 5:30
- Friendly / Cup matches in some countries (check the events page on the top Help Menu)
- Trainings after those matches

Tuesday
- Inactive user verification at 3:30
- Bosman player transfers at 10:00
- Friendly / Cup matches in some countries (check the events page on the top Help Menu)
- Trainings after those matches

Wednesday
- Stats database updates at 5:30
- League matches in some countries (check the events page on the top Help Menu)
- Trainings after those matches

Thursday
- Free player transfers at 12:00
- League matches in some countries (check the events page on the top Help Menu)
- Trainings after those matches

Friday
- Financial updates at 8:00
- Bosman transfers at 10:00
- Tournament matches at 13:00
- International Teams Competition matches at 13:00
- National Team matches at 20:00
- Tournament awards and next round draws at 15:00
- International Team Competition awards and next round draws at 15:00

Saturday
- Stats database updates at 5:30
- League matches in some countries (check the events page on the top Help Menu)
- Trainings after those matches

Sunday
- Friendly Pool match assignments at 2:30
- League matches in some countries (check the events page on the top Help Menu)
- Trainings after those matches

Updates that run on a daily basis:
- Injury updates at 5:30
- Teams assignments to new users at 6:00
- Regular transfers run every single minute

Other updates worth mentioning
- Cup awards are run on Wednesdays at 1:30
- Elections end on Sundays at 14.00 and on Fridays at the same time (for temporary elections only)
- Season transition is run during the night on the Monday after the last round of the league

These times are described using PM's server-time (Portugal/Lisbon time, normally GMT+0). Eventually, there might be some delays during these updates, so please be patient if delays do occur.
